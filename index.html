Daniel Boone National Forest Canopy Analysis: CHM & Classification

Daniel Boone National Forest was established in 1937, originally Cumberland National Forest,and is located in McKee, KY, close to London. The forest's name was changed as nod to frontiersman Daniel Boone (attributed to settling Kentucky) in 1966.

The goal of this project which I originally had done for my GEO 419 (remote sensing) project file was to see if using high resolution color near infrared imagery alogside lidar point cloud data, would allow me to estimate and classify prominent tree communties in the Daniel Boone National Forest. This took a lot of trial and error with modeling in Erdas, but eventualy i was able to create a 5 band image (RGB, Nearinfrared, and Elevation) to classify and did so with success. It's important to note that without field data to confirm, these classes are just experimental and not meant for operational use. I plan to visit DBNF this summer and test the accuarcy!
